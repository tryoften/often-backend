## Setup
Tested on node v0.12.7. Steps:

1. Install libvips: [General Instructions](https://github.com/jcupitt/libvips), [Mac OS X Instructions](http://www.vips.ecs.soton.ac.uk/index.php?title=Build\_on\_OS\_X)
2. `npm install`
